Le Wentzle

Regles du jeu:

Le wentzle est un jeu style Motus dont le but est de trouver le mot cache, un mot de l'argot francais.

  • Choisissez le nombre de lettres du mot a trouver
  • Choisissez le nombre de tentatives
  • Cliquez sur commencer pour debuter la partie
  • Vous disposez d'un bouton d'aide, utilisable une seule fois par partie, qui vous donnera une lettre
  • Vous disposez de la definition du mot si besoin
  • Les lettres rouges ne sont pas dans le mot
  • Les lettres oranges sont dans le mot a une autre place
  • Les lettres vertes sont a la bonne place
